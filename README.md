# 😂 Joke Verse

Joke Verse is a fun web application that delivers random jokes using the **JokeAPI**.  
It is built with **Node.js and Express** and deployed on **Render**.
Check Mater Node for Project Files

---

## 📌 About the Project

Joke Verse fetches random jokes from the **JokeAPI** and displays them to users through a simple web interface.

This project demonstrates:
- Using APIs in Node.js
- Building a backend with Express
- Handling HTTP requests
- Deploying Node applications on Render

---

## 🛠 Tech Stack

- Node.js
- Express.js
- JokeAPI
- HTML
- CSS
